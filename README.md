
# EventZen - Event Management System

## Project Overview
Full-stack event management application built with:
- Backend: Spring Boot
- Frontend: React
- Database: PostgreSQL
- Authentication: JWT

## Features
- User Registration
- Event Creation
- Attendee Management
- Budget Tracking

## Getting Started
1. Clone the repository
2. Set up backend
3. Set up frontend
4. Configure database
5. Run application

## Installation
### Prerequisites
- Java 11+
- Node.js 16+
- PostgreSQL
- Docker (Optional)

### Backend Setup
```bash
cd backend
mvn clean install
